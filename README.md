## AI Chatbot
A simple conversational chatbot built with OpenAI's GPT-3.5 API and Python.

**Tech Used:**
- Python
- OpenAI API
- python-dotenv for secure key storage

**How to Run:**
1. Clone the repo
2. Install dependencies: `pip install openai python-dotenv`
3. Create a `.env` file with: `OPENAI_API_KEY=your_key_here`
4. Run: `python Main.py`
5. Type 'exit' to quit
